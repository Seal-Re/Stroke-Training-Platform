# 基于大模型的康复训练平台，前端

>脑卒中认知认知康复系统

## 项目简介
脑卒中认知康复系统是一个面向脑卒中患者认知康复的应用，提供认知评估、认知训练、AI建议以及医患管理等功能。系统分为患者端和医生端两个角色，分别提供不同的功能模块。

## 技术栈
- 前端框架：Vue 3
- 状态管理：Pinia
- UI组件库：Element Plus
- 路由管理：Vue Router
- 图表可视化：ECharts
- Markdown渲染：vue3-markdown-it
- 样式处理：CSS、SCSS、Tailwind CSS

## 功能模块

### 患者端功能
1. **首页**：系统总览
2. **认知评估**：提供认知能力评估功能
3. **认知训练**：提供认知训练内容
4. **AI建议**：展示基于AI的康复建议

### 医生端功能
1. **首页**：医生工作台
2. **医患配对**：管理医生与患者的配对关系
3. **患者详情**：查看患者详细信息
4. **评估情况**：查看患者评估结果
5. **医生建议**：为患者提供专业建议
